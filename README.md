Temperature Converter ( https://www.online-python.com/DlCgJwTZLn )

Ця програма дозволяє конвертувати температуру з градусів Цельсія у градуси Фаренгейта.
Користувач вводить температуру в Цельсіях, а програма автоматично обчислює та виводить відповідне значення у Фаренгейтах за формулою:
F = (C × 9 / 5) + 32

Приклад використання

Введіть температуру в Цельсіях: 25  
Температура у Фаренгейтах: 77.0

Введіть температуру в Цельсіях: 0  
Температура у Фаренгейтах: 32.0
